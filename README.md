## 배경
추첨번호를 나눠주고 선물을 추첨하기로 했다.
큰 모니터 화면에 추첨번호를 크게 보여주기위해서 사이트를 만들어 보여주면 어떨까해서 만들어보았다.

## 사용 사이트
https://wizardly-bhabha-f2163e.netlify.app/

## 특징
 - 리액트/리덕스로 구현
 - scss 조금 사용

## 사용방법

 - 시작 숫자와 끝 숫자를 입력해서 그 사이 숫자를 입력할 수 있다.

## 문제점

 - setState가 비동기적으로 이뤄져서 중복된 숫자가 나올 때가 있다.
